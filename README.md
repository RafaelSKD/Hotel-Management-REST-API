## Hotel Management REST API

Comprehensive backend project developed during an advanced REST API course, focused on designing and implementing scalable **RESTful Web Services** using **Python** and **Flask**. The project simulates a hotel management platform, covering real-world backend development practices from API creation to deployment.

### Key Features
- Full **CRUD operations** for hotels and related resources  
- User registration with **email confirmation**  
- Secure authentication system with **login/logout**  
- Advanced hotel search and filtering endpoints  
- Relational database integration using **SQLAlchemy**  
- Support for **SQLite**, **PostgreSQL**, and **MySQL**  
- API testing and validation with **Postman**  
- Structured RESTful architecture following best practices  

### Skills Developed
- REST API design and architecture  
- Backend development with Flask  
- Database modeling and ORM usage  
- Authentication and access control  
- API testing and debugging  
- Cloud deployment and server configuration  

This project strengthened my backend engineering skills by combining API development, databases, security, and production deployment in a practical end-to-end application.
